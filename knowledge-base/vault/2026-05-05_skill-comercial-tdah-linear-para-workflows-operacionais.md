| Risco | Como Aparece | Impacto | Prevenção v2 |
|---|---|---|---|
| **Overengineering** | Scripts complexos, dependências desnecessárias | Cliente não consegue usar | Scripts Python stdlib only, zero pip install obrigatório |
| **Promessa médica TDAH** | Linguagem tipo "cura", "trata", "melhora sintomas" | Risco legal grave | Forbidden claims em tests + disclaimer em cada arquivo comercial |
| **Automação perigosa** | MCP write sem confirmação, delete sem backup | Perda de dados do cliente | Policy MCP obrigatória + confirmation gates em scripts |
| **Skill redundante** | Overlap com skills nativas Anthropic | Cliente confuso sobre quando usar | SKILL.md seção "Quando NÃO usar" explícita |
| **Curso grande demais** | 50 módulos antes de validar mercado | Tempo perdido, produto não vendável | MVP: 10 módulos max, exercícios curtos, templates prontos |
| **Linear virar depósito** | Issues sem DoD, backlog infinito | Sistema vira ruído | Issue template obrigatório com DoD field, validate_dod.py |
| **Falta de DoD claro** | Tarefas "quase prontas" infinitas | Cliente nunca fecha sprint | DoD checklist em cada workflow + visual no dashboard |
| **Claims Anthropic/Linear** | "Método oficial", "certificado por" | Risco legal, perda credibilidade | Disclaimer explícito + test scan de "oficial", "certificado", "endorsed" |
| **Pivô no meio da sprint** | Cliente muda objetivo no dia 3 | Sprint falha, frustração | Parking Lot obrigatório + "do_not_pivot" label + review semanal |
| **Dados sensíveis via MCP** | Conectores com acesso PII sem avisar | GDPR violation, perda confiança | MCP policy com data classification + user consent template |
| **GitHub publish sem license** | Repo público sem definir termos | Uso indevido, zero proteção IP | Adicionar LICENSE (MIT padrão) + copyright header |
| **Pricing não validado** | Preço alto demais ou baixo demais | Zero vendas ou receita insuficiente | Pricing tiers template + competitor analysis section |
| **Sem lead magnet** | Cliente não sabe onde começar | Zero conversão | Checklist gratuito "5 Sinais de Caos Operacional" |
| **Documentação só PT-BR** | Mercado internacional inacessível | Limite growth Holland | README bilíngue + key docs EN |
| **Scripts sem testes** | Bug em produção no cliente | Reputação perdida | Pytest suite obrigatório + CI GitHub Actions |
