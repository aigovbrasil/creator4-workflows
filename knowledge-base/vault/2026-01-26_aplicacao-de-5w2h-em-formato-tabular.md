**Claude** _2026-01-26_

Para converter este Markdown em PDF, você tem algumas opções:

**Opção 1 - Online (Mais Fácil):**
1. Copie o conteúdo do artefato
2. Cole em: https://www.markdowntopdf.com/ ou https://md2pdf.netlify.app/
3. Clique em "Convert" e baixe o PDF

**Opção 2 - VS Code (Recomendado):**
1. Cole o conteúdo em um arquivo `.md` no VS Code
2. Instale a extensão "Markdown PDF"
3. Clique direito → "Markdown PDF: Export (pdf)"

**Opção 3 - Pandoc (Terminal):**
```bash
# Instale pandoc
brew install pandoc  # macOS
# ou
sudo apt install pandoc  # Linux

# Converta
pandoc nexus-guide.md -o nexus-guide.pdf
```

Qual método prefere? Posso ajustar o formato do Markdown para melhor conversão em PDF se necessário!
