   155	## How to use this repo
   156	
   157	**To install a skill in Claude.ai:**
   158	1. Download the `.skill.zip` file from the relevant category folder
   159	2. Go to claude.ai → Settings → Skills → Add skill
   160	3. Upload the `.skill.zip` file
   161	4. The skill is now active in your account
   162	
   163	**To use the batch builder:**
   164	```bash
   165	python tools/batch-skill-builder/skill_builder_batch.py \
   166	  --input ./skills/[category] \
   167	  --output ./reports/batch/[run-name] \
   168	  --overwrite
   169	```
   170	
   171	**To find the right skill:**
   172	→ See trigger phrases in each skill's README, or consult `x-ray-self-knowledge` for X-Ray suite routing.
