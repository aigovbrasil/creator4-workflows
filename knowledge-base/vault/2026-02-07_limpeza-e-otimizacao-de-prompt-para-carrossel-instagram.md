## ✅ Soluções Técnicas

### **Para SVG:**
```svg
<g id="box-01">
  <!-- Container com clipping -->
  <clipPath id="clip-box-01">
    <rect x="210" y="280" width="400" height="160" rx="8"/>
  </clipPath>
  
  <!-- Background -->
  <rect x="210" y="280" width="400" height="160" rx="8" fill="#f8f9fa"/>
  
  <!-- Texto com foreignObject (permite quebra automática) -->
  <foreignObject x="230" y="300" width="360" height="120" clip-path="url(#clip-box-01)">
    <div xmlns="http://www.w3.org/1999/xhtml" style="
      font-family: 'Merriweather', serif;
      font-size: 17px;
      line-height: 1.5;
      color: #2c3e50;
      overflow: hidden;
      text-overflow: ellipsis;
    ">
      Prioridade absoluta para estudos peer-review
    </div>
  </foreignObject>
</g>
```

### **Para HTML/CSS:**
```css
.box {
  width: 400px;
  height: 160px;
  padding: 20px; /* Espaço interno */
  box-sizing: border-box; /* Inclui padding na largura total */
  overflow: hidden; /* Esconde overflow */
  border-radius: 8px;
}

.box-text {
  font-size: 17px;
  line-height: 1.5;
  
  /* Limita a 3 linhas com reticências */
  display: -webkit-box;
  -webkit-line-clamp: 3;
  -webkit-box-orient: vertical;
  overflow: hidden;
  text-overflow: ellipsis;
}
```
