## Especificações para Replicar e Personalizar

**Sistema de Hierarquia Tipográfica**

Para criar um sistema similar adaptado à sua marca, você necessita definir três categorias principais de texto. A categoria Subtítulo deve empregar fonte com peso semibold ou bold, tamanho aproximado de dezoito a vinte e dois pixels, adequada para títulos de seção e cabeçalhos secundários. A categoria Corpo representa o texto padrão para conteúdo geral, utilizando peso regular ou medium, com tamanho entre quatorze e dezesseis pixels, otimizado para leitura prolongada. A categoria Estilo Fixo emprega fonte monospace para código, dados técnicos ou conteúdo que requer alinhamento preciso de caracteres.

**Controles de Formatação**

O sistema de formatação inline deve disponibilizar quatro opções fundamentais organizadas em linha horizontal. Bold incrementa o peso da fonte para destacar informações importantes sem alterar o tamanho do texto. Itálico aplica inclinação para ênfase sutil, citações ou termos estrangeiros. Underline adiciona linha inferior para destacar links ou informações críticas, embora deva ser utilizado com moderação para evitar confusão com hiperlinks. Strikethrough aplica linha atravessada para indicar conteúdo removido ou desatualizado, particularmente útil em listas de tarefas.

**Sistema de Cores para Texto**

A paleta de cores observada nas imagens sugere um sistema com pelo menos três cores distintas além do texto padrão. A primeira cor (dourado/amarelo com valor aproximado hex #D4A317 ou similar) serve para destacar informações primárias ou títulos importantes. A segunda cor (azul com valor aproximado hex #3B82F6) funciona para elementos secundários, links ou categorização alternativa. A terceira cor (rosa/vermelho com valor aproximado hex #FF1493 ou similar) pode indicar urgência, alertas ou categorização adicional.

**Controles de Lista e Organização**

O sistema deve incorporar três tipos de listas acessíveis através de botões dedicados. Listas com bullets utilizam marcadores circulares para itens não ordenados, ideal para enumerações sem hierarquia específica. Listas numeradas empregam numeração sequencial automática para procedimentos, instruções passo a passo ou conteúdo hierárquico. Checklists apresentam caixas de seleção clicáveis para tarefas e itens que requerem acompanhamento de conclusão.

**Controles de Alinhamento**

Os controles de alinhamento devem oferecer três opções padrão. Alinhamento à esquerda serve como padrão para a maioria dos textos, proporcionando leitura natural em idiomas ocidentais. Alinhamento à direita acomoda necessidades específicas de design ou idiomas que utilizam escrita da direita para esquerda. Alinhamento centralizado funciona para títulos, citações destacadas ou elementos que requerem simetria visual.

**Ferramentas Complementares**

A interface deve incluir ferramentas adicionais posicionadas na barra superior. Controle de desenho permite anotações manuais e markup visual. Seletor de cor oferece paleta completa para personalização além das cores predefinidas. Ferramenta de anexo possibilita inserção de imagens, documentos e outros arquivos. Controle de formatação de parágrafo ajusta espaçamento e indentação.

## Adaptação para Seu Brand Kit

Para personalizar este sistema tipográfico conforme sua identidade visual, você deve mapear cada elemento às especificações da sua marca. Substitua as cores observadas (dourado, azul, rosa) pelas cores primária, secundária e de destaque do seu brand kit, mantendo os códigos hex precisos para consistência. Defina suas fontes aprovadas para cada categoria: uma fonte display ou sans-serif moderna para Subtítulo, sua fonte de corpo padrão para texto Corpo, e uma fonte monospace legível para Estilo Fixo.

Estabeleça a escala tipográfica exata alinhada ao seu sistema de grid base, garantindo que todos os tamanhos sejam múltiplos do seu grid unit. Documente os pesos de fonte disponíveis em sua família tipográfica e mapeie-os aos estados de formatação (regular, medium, semibold, bold). Verifique todos os contrastes de cor contra fundos claros e escuros para garantir conformidade WCAG AA no mínimo.

Configure os valores de line-height apropriados para cada categoria de texto, tipicamente um vírgula cinco para corpo de texto e um vírgula dois para títulos. Defina o espaçamento entre parágrafos e margens ao redor de listas para manter consistência visual. Estabeleça a hierarquia de cores para estados interativos como hover, active e disabled em todos os controles de formatação.

Esta estrutura permitirá que você replique a funcionalidade e organização visual do aplicativo de notas enquanto mantém total alinhamento com sua identidade de marca e design system estabelecido.
