# Leitura do Projeto — Visão de Designer

Entendi o escopo completo. Antes de construirmos o prompt, veja minha leitura estruturada:

**Objeto físico:** Prisma triangular (tipo totem de mesa), 3 faces de 90cm × 10cm cada. Provavelmente impresso em adesivo vinílico, papel couché ou PVC rígido, fixado nas faces de um suporte triangular.

**O maior desafio de design aqui é densidade vs. legibilidade.** 90cm de largura com 10cm de altura é uma proporção muito panorâmica e estreita — precisa de uma hierarquia visual forte, tipografia pequena mas legível, e grid rígido por zonas.

---

**Face 1 — Identidade & Operação Pessoal**
Você quer: card de dados pessoais + rotina diária/semanal/mensal + calendário Q1 + metas Uber + concursos + QR Code. Isso é muita informação para 90×10cm. A solução é zoneamento horizontal em blocos bem demarcados com cores e ícones como separadores visuais.

**Face 2 — Sistema de Estudos**
Fluxo de estudos + roadmap + checklist + QR Codes de portais + leituras indicadas. Aqui o conceito visual é linha do tempo horizontal com ramificações — funciona bem no formato panorâmico.

**Face 3 — Projeto TDHA / ONG / Loja**
Três pilares (Doc → Vídeo → Distribuição) + produtos + afiliados + QR Codes. Precisa de uma estrutura de kanban visual ou funil horizontal com status de progresso.
