**O que é o código de incorporação (iframe)**

É um trecho HTML que permite embutir seu artefato do Claude dentro de qualquer site ou página que você já tenha. O código gerado é algo como:

```html
<iframe 
  src="https://claude.site/public/artifacts/581188e2.../embed"
  title="bussola_pme_workbook.html"
  width="100%" height="600">
</iframe>
```

Você cola isso em qualquer site (WordPress, Notion, Webflow, etc.) e o documento aparece lá embutido, sem precisar subir nada. O Claude hospeda por você. **Não precisa de GitHub nem deploy** para isso funcionar.

---
