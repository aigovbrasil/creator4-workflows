---

## PROMPT PARA ENVIO

```
Você é um especialista em gestão de projetos para empreendedores solo com TDAH, especializado em criar planos executáveis e neuroadaptados.

# CONTEXTO DO SOLICITANTE
- Nome: Leonardo, 30 anos
- Perfil: TDAH + superdotação, empreendedor solo
- Conhecimentos: Neuromarketing, design de produto, modelos de negócio, IA
- Limitações: Design criativo, programação, gestão de projetos, mídias digitais
- Ferramentas disponíveis: Agentes de IA (Manus Agent para automação)

# OBJETIVO
Criar um documento único, imprimível, datado e acionável com planejamento de 9 sprints semanais (25/01/2026 a 01/04/2026) para estruturação completa de um ecossistema multicanal de produtividade ADHD.

# ESCOPO DO ECOSSISTEMA

## Ferramentas SaaS (3 produtos):
1. **Executar** (pronta) - necessita go-to-market e beta users
2. **Estruturar** (ex-Safe Protocol) - blueprint pronto, falta desenvolvimento web
3. **Entender** (ex-Academic Flow) - falta aplicação completa
4. **Creator** (bônus) - automação de conteúdo multicanal via one-shot

## Canais (ordem de prioridade):
1. LinkedIn (autoridade principal)
2. TikTok
3. YouTube
4. Instagram

## Plataformas complementares:
- Blog (artigos acadêmicos + conteúdo)
- Marketplace infoprodutos
- Marketplace produtos físicos (Loja do TDH)
- ONG integrada ao portal

## Modelos de monetização:
- AdSense (blog/vídeos)
- Assinaturas SaaS
- Consultoria/gestão de projetos
- E-books e cursos online
- Produtos físicos
- Programas de afiliados
- Personalização de projetos (GPT/Gemini)

# PREMISSAS TÉCNICAS
- Priorizar tecnologias que acelerem processos
- Máxima eficiência via workflows automatizados (Manus Agent)
- Estratégia de reaproveitamento: 1 conteúdo → múltiplos formatos
- Identidade visual única para todos os materiais
- Integrações automáticas (e-mail, apps)

# TAREFAS P0 (CRÍTICAS):

2. Configurar perfil LinkedIn (migrar diplomas, completar cursos: Agentes IA + Gestão de Projetos)
3. Definir workflows eficientes antes dos sprints

5. Validar ONG com profissional de saúde + entrada no cartório

# ENTREGÁVEIS JÁ PRONTOS:
```
