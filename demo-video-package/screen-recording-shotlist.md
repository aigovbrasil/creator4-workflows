# Screen Recording Shotlist

> Gravação a 1920×1080, 60fps, cursor enlarged.
> Navegador limpo: sem extensões visíveis, sem bookmarks bar, tema GitHub light.
> Janela em modo "Frame" do Screen Studio (margem suave, sombra leve, fundo neutro `#F6F4EF`).

---

## Cena 1 · Cold open — abas vazias
**Duração:** 0:00 – 0:08 (~8s)
**Mostrar:** navegador com 4 abas abertas — Claude, ChatGPT, Gemini, Notion. Todas em "New chat" / doc em branco. Cursor pulando.
**Falar:** *"Toda segunda-feira você abre uma conversa nova com o Claude."*
**Zoom / crop:** plano geral, sem zoom. Movimento de cursor faz o olho.
**Texto na tela:** `Every Monday. New chat. Same 20 minutes of context.` — canto inferior esquerdo, lower third.

---

## Cena 2 · Dor concreta — três frames rápidos
**Duração:** 0:08 – 0:25 (~17s)
**Mostrar:**
1. Doc Notion bagunçado com headings "prompts antigos", "ideia março", "rever depois";
2. Busca `Ctrl+F` por "decisão" — 0 resultados;
3. Dois posts LinkedIn lado a lado, claramente em vozes diferentes.
**Falar:** narração do bloco "10–25s" do script.
**Zoom / crop:** corte duro entre os três. Cada frame ~3s. Zoom de 110% no segundo frame na barra de busca.
**Texto na tela:**
- Frame 1: `Copy-pasted prompts.`
- Frame 2: `Lost decisions.`
- Frame 3: `Inconsistent voice.`

---

## Cena 3 · Tese — chat vs system
**Duração:** 0:25 – 0:45 (~20s)
**Mostrar:** card animado (pode ser slide HTML, não precisa ser tela real).
- Esquerda: caixa "Chat" solta no centro.
- Direita: caixa idêntica dentro de moldura "System" com 4 labels — `Context · Workflows · Decisions · Templates`.
**Falar:** bloco "25–45s" do script.
**Zoom / crop:** plano fixo. A moldura "System" desenha-se com animação de stroke (1.5s).
**Texto na tela:** `The problem isn't the prompt. Your work isn't a chat. It's a routine.`

## Cena 4 · GitHub — tela inicial do repo

## Cena 10 · Before / After (split screen)
**Duração:** 1:20 – 1:40 (~20s)
**Mostrar:** split screen.
- Esquerda: chat genérico do Claude, briefing colado de 400 palavras, output morno.
- Direita: mesmo Claude — o usuário digita só `start` — Claude responde com rascunho já estruturado, voz reconhecível.
**Falar:** bloco "75–100s" do script.
**Zoom / crop:** começa em 50/50. Após 8s, zoom in no lado direito para 70/30, destacando a primeira linha do output ("rascunho já tá na sua voz").
**Texto na tela:**
- Esquerda (sticky 4s): `Before: paste briefing. Wait. Rewrite half.`
- Direita (sticky 4s): `After: type "start". Claude already knows.`

## Notas técnicas de gravação

| Item | Especificação |
|---|---|
| Resolução | 1920×1080 |
| Framerate | 60fps |
| Áudio | Mono, -14 LUFS, sem música de fundo (ou bed muito sutil ≤ -28 LUFS) |
| Cursor | Highlight on click, sem trail |
| Browser | Tema light, zoom de página 100%, sem extensões visíveis |
| Janela | Frame Screen Studio, margem 80px, sombra leve |
| Fonte da janela | tamanho normal do sistema |
| Transições | Cortes duros. Apenas fade no início/fim e nos cards "tese" e "CTA" |

## O que NÃO mostrar
- Nenhuma chave de API, token, email pessoal.
- Nenhum arquivo `.env` ou config sensível.
- Nenhum nome de cliente real em arquivos abertos.
- Nada que pareça "produto fechado" — deixar visível que é um repo público em evolução.
