# On-Screen Copy

> Todas as frases que aparecem **na tela** durante o vídeo.
> Não confundir com narração (que está em `120-second-script.md`).
> Inglês curto, alto-contraste, máx. 7 palavras por linha.

---

## 1 · Problema (0–10s)

```
Every Monday.
New chat.
Same 20 minutes of context.
```

Variante curta (caso o frame fique apertado):
```
Every Monday. Same 20 minutes lost.
```

## 10 · Regras de copywriting on-screen

1. Verbos no presente. Sem futuro.
2. Sem ponto de exclamação. Nunca.
3. Sem emoji. Nunca.
4. Sem maiúsculas em frase inteira (exceto eyebrows `LAYER 01`).
5. Se uma frase precisa de 3 linhas, está errada. Reescreve.
6. "Você" só na narração — on-screen é impessoal.
7. Nunca prometer resultado quantitativo além de "20 min / Monday" (essa é a única métrica autorizada, e é honesta).
